# Trabalho Final – Banco de Dados II

Este repositório contém o trabalho final da disciplina **Banco de Dados II**, organizado em três etapas independentes, cada uma abordando um tema específico:

- **Etapa 1:** conexão e consultas em PostgreSQL.  
- **Etapa 2:** CRUD em Python com PostgreSQL + Redis (cache).  
- **Etapa 3:** uso de **XQuery** e integração de XML com PostgreSQL.
